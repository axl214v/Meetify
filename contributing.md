First of all, thank you for your interest in improving Meetify! We welcome contributions from the community, whether they are bug fixes, documentation improvements, or new features.

## How to Contribute

### 1. Reporting Bugs
- Use GitHub Issues to report bugs.
- Provide a clear description of the issue and steps to reproduce it.
- Attach logs or screenshots if possible.

### 2. Pull Requests (PRs)
- Fork the repository and create your branch from `main`.
- Ensure your code follows the existing style (Node.js/Vanilla JS).
- Write clear, concise commit messages.
- Update the README if you've added new features or changed configurations.

### 3. Feature Requests
If you have a big idea, please open an Issue first to discuss it with the maintainers before starting any work.

## Code of Conduct
Please be respectful to other contributors. We aim to build a supportive and professional community.
